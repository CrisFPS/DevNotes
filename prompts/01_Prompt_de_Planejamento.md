# Prompt de Planejamento

Atue como um especialista em Engenharia de Software, Planejamento de Projetos de Software, SDLC e uso de Inteligência Artificial Generativa como apoio ao desenvolvimento.

Quero planejar um pequeno aplicativo chamado DevNotes Local.

## Contexto do projeto:
O DevNotes Local será uma aplicação web local, simples e didática, criada para fins de estudo em uma especialização de Engenharia de Software com Automação e Inovação usando Inteligência Artificial Generativa.

O objetivo da aplicação é permitir que eu organize, pesquise e reutilize snippets, SQLs, anotações técnicas, pequenos scripts, exemplos de código, arquivos exportados de sistemas legados e regras de negócio.

O sistema será usado apenas localmente, sem publicação em produção, sem autenticação, sem múltiplos usuários e sem infraestrutura em nuvem.

O MVP deve ser pequeno, viável para desenvolvimento assistido por IA em aproximadamente 3 a 12 horas, mas completo o suficiente para praticar as principais fases do SDLC.

## Tecnologias obrigatórias:
- Python 3.11;
- FastAPI;
- SQLite;
- SQLAlchemy;
- SQLite FTS5;
- Jinja2;
- HTML simples;
- CSS simples;
- JavaScript simples apenas se necessário;
- Highlight.js;
- pytest;
- Git;
- ambiente virtual local com venv.

# Funcionalidades mínimas esperadas:
- cadastrar conteúdos técnicos;
- editar conteúdos;
- excluir conteúdos;
- pesquisar conteúdos;
- colar manualmente trechos de código ou texto;
- fazer upload simples de um arquivo por vez;
- visualizar conteúdos preservando a formatação original;
- visualizar código com destaque de sintaxe;
- classificar conteúdos por categoria, linguagem, tags, tipo de arquivo, sistema e domínio;
- marcar conteúdos como regra de negócio.

# Escopo fora do MVP:
- autenticação;
- múltiplos usuários;
- permissões;
- backup automático;
- APIs externas;
- frontend sofisticado;
- microsserviços;
- publicação em produção;
- arquitetura distribuída;
- versionamento interno avançado de snippets.
